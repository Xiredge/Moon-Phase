**MoonPhase**

This website finds the Moon’s phase for any given date using the NASA DE421 ephemeris and Skyfield's Library.

Visit the live website [here](https://moon-phase-063a.onrender.com)

**Screenshots of the website:**

![First Page Load](./screenshot1.png)

**Features**

- Finds the Moon’s phase for any selected date
- Illumination percentage
- Elongation angle
- Uses high-precision astronomical data

**Astronomical Data Source**

This project uses NASA JPL’s DE421 planetary ephemeris, which provides precise positional data for:
- The Moon
- The Earth
- Other celestial bodies

Ephemeris data is provided by NASA’s Jet Propulsion Laboratory through the
NASA Jet Propulsion Laboratory (JPL).

**Calculation Engine**

All astronomical calculations are performed using the
Skyfield library.

Skyfield computes accurate planetary positions using official NASA ephemeris files rather than simplified algorithms.

If you'd like to learn more about Skyfield, visit their official website:
[https://rhodesmill.org/skyfield/]
